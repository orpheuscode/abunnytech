## Open risks

1. **TikTok DM API** — requires Partner API access; only mock works today.
2. **browser-use agent wiring** — `BrowserUseProvider` stubs are present but LLM + Playwright config is not wired. Assign to whoever handles browser-use credentials.
3. **Rate limiter enforcement** — `RateLimitConfig` is defined and respected in settings but the actual token-bucket enforcement (tenacity decorators) is not yet applied to live provider calls. Fine for demo; add before production load.
4. **File upload for TikTok/Instagram** — media chunked upload is stubbed with TODO markers. Direct URL posting works via media_url field today.
5. **Shopify base URL** — requires `SHOPIFY_STORE_DOMAIN` env var; not set from `PlatformConfig.api_base_url` yet.
