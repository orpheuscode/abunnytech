- **Python models:** `packages/contracts/src/pipeline_contracts/`

## Frozen handoff objects

All of the following use Pydantic v2 with `extra="forbid"` unless noted. Unknown JSON keys are rejected so workers and APIs fail fast on typos.

### `IdentityMatrix`

Stable persona and production profile for one creator lane.

| Field | Required | Notes |
|--------|-----------|--------|
| `matrix_id` | yes | Unique id |
| `display_name` | yes | Human-facing name |
| `niche` | yes | Vertical / audience label |
| `persona` | yes | `PersonaAxis` (tone, topics, avoid_topics, optional disclosure_line) |
| `avatar` | yes | `AvatarPackRef` |
| `voice` | yes | `VoicePackRef` |
| `platform_targets` | no (default) | List of `PlatformTarget` enum values |

### `VideoBlueprint`

| Field | Required | Notes |
|--------|-----------|--------|

| `matrix_id` | yes | Links to `IdentityMatrix` |

| `outline` | no | Beat list |
| `suggested_caption` | no | Default `""` |
| `hashtags` | no | List of tags (contract does not strip `#`) |
| `audio_id` | no | Links to trending audio when set |
| `duration_seconds_target` | no | Default `15`, range 1–600 |

Outcome of posting (or attempting to post) a package.

| Field | Required | Notes |
|--------|-----------|--------|
| `record_id` | yes | Unique id |

### `OptimizationDirectiveEnvelope`

Versioned feedback for one or more stages. Outer shape is fixed; inner payload is opaque.

| Field | Required | Notes |
|--------|-----------|--------|
| `directive_id` | yes | Unique id |
| `target_stages` | no | List of `DirectiveTargetStage` |

| `rationale` | no | Human-readable |

Consumers must inspect `envelope.schema_version` before interpreting `envelope.payload`.

## Versioning rules (summary)

- Bump **`pipeline-contracts`** (PEP 440) on any observable contract change.
- **Patch:** docs / JSON Schema metadata only; same acceptance for existing payloads.
- **Minor:** backward compatible (new optional fields, new enum members, relaxed validation).
- **Major:** breaking (rename/remove fields, remove enum values, stricter validation, `extra` policy changes).
- For **nested** directive data, bump `Envelope.schema_version` when `payload` semantics change.

Helpers: `pipeline_contracts.versioning.package_version()`, `DEFAULT_ENVELOPE_SCHEMA_VERSION`.
