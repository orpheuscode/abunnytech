## Quick Start

```bash
pip install -e ".[dev]"
uvicorn apps.api.main:app --reload
# POST http://localhost:8000/api/seed to load demo data
```

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/health` | Health check |
| GET | `/api/collections` | List all entity collections |
| POST | `/api/seed` | Insert demo fixture data |
| GET | `/api/{collection}` | List items (paginated) |
| GET | `/api/{collection}/count` | Count items |
| GET | `/api/{collection}/{id}` | Get item by ID |
| POST | `/api/{collection}` | Create item |
| PUT | `/api/{collection}/{id}` | Update item |
| DELETE | `/api/{collection}/{id}` | Delete item |

Collections: `identity-matrix`, `video-blueprints`, `content-packages`,
`distribution-records`, `performance-metrics`, `optimization-directives`,
`redo-queue`, `product-catalog`, `brand-outreach`, `dm-conversations`,
`trending-audio`, `competitor-watchlist`
