# m3t2 -- Dashboard / Operator UI

**Owner:** m3t2  
**Status:** DONE  
**Branch:** `feat/m3t2-dashboard`

## What shipped

- Flask owner dashboard at `runtime_dashboard/`
- 6 pipeline stage pages (Identity, Discovery, Content, Distribution, Analytics, Monetization)
- Demo control surface with dry-run simulation, artifact inspector, and directive/redo viewer
- Guided step-by-step demo walkthrough (8 steps covering the full pipeline story)
- Fixture JSON with seeded demo data covering all contract types
- API client that falls back gracefully from live API to fixture data

## How to run

```bash
uv run python -m runtime_dashboard.flask_owner_app
```

Dashboard loads on `http://localhost:8501` by default (configurable via `DASHBOARD_PORT`).
